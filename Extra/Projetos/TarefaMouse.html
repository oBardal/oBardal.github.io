Qual a função de cada peça?

Cada peça no mouse tem uma função diferente, e algumas trabalham em conjunto para terem mais precisão em algumas medidas!


  • Botões

    Os botões do mouse servem para cada um registrar os inputs do usuário, o terceiro botão serve para registrar quando o usuário prime a roda do mouse



  • Hardware Simples

    O Hardwave usado serve para fazer pequenos calculos para registrar a movimentação x e y do aparelho e enviar corretamente ao computador



  • Emissor / Receptor de luz Infra-Vermelho

    Os dois sensores servem para identificar quando a roda, ou "Scroll" gira e manda a informação de "True" se o sinal é visivel e "False" se o contrário, além de que se está girando para frente ou para trás



  • Lente / Luz LED

    Os dois servem para auxiliar a leitura do hardware com o movimento do mouse



  • Carcassa plástica

    As duas peças que são presas por 1 parafuso e um pouco de engenharia servem para dar conforto e um visual mais bonito e funcional ao mouse



Observações:

  • Matéria-Prima usada:

    Plástico é comumente usado na maior parte dos eletrodomésticos, possui ~60 gramas de plástico usado na produção do mouse, contando com pequenos suportes e o encapamento do fio



  • Produção:

    Cada peça é produzida de forma independentemente e depois é juntada e soldada formando um único produto



  • Só mais uma coisa:

    O mouse usado já não funcionava mais, oque foi bom pois não consegui monta-lo novamente desde que algumas pequenas peças de ferro foram ejetadas para longe, muito obrigado por ler até o final prof!
